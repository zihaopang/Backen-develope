# Java的可移植性
所谓的可移植性指的是同一个程序可以在不同的操作系统上面部署，这样就减少了开发的难度，可以执行主要依靠（JVM），JVM是一个由软件和硬件模拟出来的计算机，这样就可以实现可移植性。
